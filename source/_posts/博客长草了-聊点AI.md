博客四年没更，没什么特别原因，就是懒得写、不想维护代码。

这几年经历了不少：23 年在长沙玩拖拉拽业务，24 年在上海给酒吧写小程序，25 年自己创业，26 年到现在在外学跨境电商。

之前 AI 刚出来时我也试过，觉得没什么需求，都是伪需求。最早那些 agent 工具出来，我当时就知道普通人根本没需求去用——事实证明我是对的，最赚钱的是帮别人安装的那些人。

直到今年去学跨境电商，真有需求了，一下午就弄好了。

OpenAI 确实牛逼。思路很简单：直接装 Codex，给权限让它帮我装 Claude Code CLI，再把模型切到 DeepSeek v4 接口，省了不少钱。以前不想主动学的 Python 脚本，现在给 agent 一个思路，五分钟就写出来了。

技术提升最直观的体现是博客的 SEO。22 年那会儿我搜了半天教程去注册配置，搞了一下午没搞好，最后放弃了。今年 agent 几分钟就解决了，效果还比我折腾半天的好。

我是什么都会点，但什么都不精，AI 刚好补上了这个短板。跟领导一样，领导不一定要精通技术，但得知道有这个东西才能解决问题。技术提升了至少 50%，这算是我的优势吧。

网上有个梗说"文科生有了 agent 吊打理科生"，当个笑话看看就好。不过确实有一堆人分享自己用 AI 开发的 App、小程序，其实没啥业务逻辑也没有合理的交互设计。AI 只能顺着你的指令来，你得先知道自己要什么——就像你想写东西，起码得知道有笔有纸这回事。

对我来说，Claude 的记忆里全是我的个人信息，它就是我的个人管家。Claude Code 帮我写脚本、干杂活、维护博客——现在博客的版本升级、美化优化全是它干的。包括你现在看到的这篇文章，也是 CC 搭 DeepSeek 帮我写的。

后续有时间想拿 AI 来反编译一些小游戏，搭建个人 NAS 也有帮助。

回头看 AI 这几年的进化：三年前能说话，两年前能看东西，一年前有手了，很快就会有脚。等到那时候，可以重仓机器人了——我个人是这么判断的。

挺期待智械危机的（开玩笑）。

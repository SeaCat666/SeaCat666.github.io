Manjaro双系统安装（折腾）记录

想着在电脑上装两个系统感觉很炫，跟朋友一起装了个deepin20v，然后换成manjaro kde了

算是腾出60G来装一个玩具 XD

准备工作

  • 叫上已安装好了的hxd辅助你完成安装
  • 腾出60G装系统（此电脑右击》属性》磁盘管理》右击盘》压缩60G）
  • 下载manjaro镜像 下载点我（kde版）←官网下载很慢，这里贼快
  • 制作启动盘工具 下载点我

开始安装

  1. 使用balenaEtcher安装到优盘（优盘最好是空的，不然文件乱七八糟）

  2. 重启进入BIOS 选择优盘启动

  3. 分区 文件系统：ext4 挂载点：/

    看别的大佬设置比我多，但问题不大，我缺一个300MB的EFI分区，应该是硬盘问题，自己创一个就好了 创建参考

  4. 安装引导程序(踩坑)

    这里选第二个就好了，安装是在这里

    我这是有两个硬盘，第一个是C盘的系统盘，好兄弟说别选第一个，会报错

    第三个系统分区不知道是什么然后

    BIOS里只有两个硬盘一个不知道是什么，都不能进入安装的manjaro（害我重装三次 淦）

  5. 配置软件源

    去终端安装编辑器vim：sudo pacman -S vim

    更改国内镜像源：sudo pacman-mirrors -i -c China -m rank

    同步（刷新）：sudo pacman -Syy

    增加中文社区的源，在 /etc/pacman.conf 中添加 archlinuxcn 源：
    sudo vim /etc/pacman.conf

    在最后添加：

  6. 差不多了可以用了（其实并不是）

    意思就是可以开始折腾了，安装软件就可以折腾很久了（输入法安装一下午 淦）

几个小指令

  1. 安装软件命令：
    sudo pacman -S 软件名字
  2. 卸载软件命令：
    sudo pacman -R 软件名字
  3. 查看系统信息：
    screenfetch
